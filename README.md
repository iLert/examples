# Using iLert terraform provider with grafana

This example related to [this blog post](...)

### Usage

```sh
export ILERT_ORGANIZATION=...
export ILERT_USERNAME=...
export ILERT_PASSWORD=...

docker-compose up -d
terraform init
terraform apply
```
